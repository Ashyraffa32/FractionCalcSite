Frequently Asked Questions

Google Play Protect block this app and said this is a "malware". is that true??

No, no. it was a false positive because we distribute our app independently. You can check the source code on our GitHub repo here to see if our app was safe or not.

Windows Defender flagged this software. it is safe??

It was a false positive because our app wasnt signed with a certificate. the app is open-source so you can see the code for yourself.

Is FractionCalc open-source?

Yes! You can view and contribute to the source code on our GitHub repository.

Does FractionCalc require an internet connection?

No. FractionCalc works completely offline, so you can use it anywhere, anytime.

Who made FractionCalc?

FractionCalc was made by Ashyraffa. It’s built to make math easier and more approachable, without ads or distractions.

Where can I download FractionCalc?

You can download FractionCalc for your Windows, Linux or Android device from the Downloads page.
